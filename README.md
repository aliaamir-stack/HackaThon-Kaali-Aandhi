## 🎯 About the Hackathon

**AI Mustaqbil 2.0** is a **Global Youth Agentic AI Hackathon** focused on building next-generation **multi-agentic AI systems**.

Participants will design and develop autonomous AI pipelines where specialized agents **think, plan, debate, and collaborate** to solve meaningful real-world challenges.

Whether you're a student, developer, researcher, or AI enthusiast, this is your opportunity to:
- Explore advanced **agentic workflows**
- Create intelligent collaborative AI systems
- Push beyond single-agent limitations
- Learn from mentors and connect with the AI community

**Format**: Hybrid (Online + Onsite at NASTP Karachi)  
**Duration**: 24-hour hackathon (March 27–28, 2026)

## 🚀 Quick Start for Participants

1. **Register** for the hackathon (check official announcements)
2. **Read the challenges** → [`AI_Mustaqbil_2_Challenges.pdf`](AI_Mustaqbil_2_Challenges.pdf)
3. **Review evaluation criteria** → [`AI_Mustaqbil_2_Evaluation_Criteria.pdf`](AI_Mustaqbil_2_Evaluation_Criteria.pdf)
4. **Build your multi-agent system** using any tech stack you prefer
5. **Submit** your project before the deadline (submission link will be shared during the event)

### Recommended (but not mandatory) Tech Stack:
- Python
- LangChain / LangGraph / CrewAI / AutoGen / LlamaIndex
- Any LLM provider (OpenAI, Groq, Anthropic, Gemini, local models, etc.)
- Vector stores, tools, memory, and agent orchestration libraries

---

## 🏆 Evaluation Criteria (Summary)

Full details are available in the official PDF. Projects will be judged on:

- **Innovation & Creativity**
- **Technical Depth** of the multi-agent architecture
- **Quality of Agent Collaboration** (planning, reasoning, tool use, handoffs)
- **Real-world Impact** and problem-solving effectiveness
- **Code Quality, Documentation & Maintainability**
- **Demo & Presentation**
